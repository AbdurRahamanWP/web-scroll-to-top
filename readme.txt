A customizable Web Scroll To Top plugin for your WordPress site, allowing you to display a Scroll To Top screen until the site is fully Scroll.

== Description ==

** web-scroll-to-top** is a simple and lightweight WordPress plugin that adds a customizable Website Scroll To Top to your website.  Web-Scroll-To-Top provide a user-friendly experience by showing a Button screen while your website's Scroll To Top position. You can customize the appearance, select a preset image , and control where the Scroll Top displays on your website .

**Key Features**:
– Multi Type Scroll To Top option
– Customize text, font size, font color, and background color of the preloader
– Select from preset images.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/web-scroll-to-top` directory or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to **Settings** -> **  Scroll To Top Settings ** to configure the plugin.
4. Save your settings and the  Scroll To Top  will appear on the pages you specify.
